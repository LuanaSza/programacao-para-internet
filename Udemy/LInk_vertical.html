HTML

O que é ?
HTML (acrônimo para a expressão inglesa HyperText Markup Languagem, que significa linguagem de marcação de Hipertexto) é
uma linguagem de marcação utilizada para produzir páginas na Web. Documento HTML podem ser interpretados por navegadores
Sobre o curso;
Este curso ensina desde os conceitos básicos da linguagem HTML, até a criação de formulários para a Web, ideal para
iniciantes, o par quem quer um guia de consulta, também possui um Glossário com todas as palavras mais utilizadas na Internet

DreamWeaver

O que é?
O macromedia DreamWeaver é um software de criação para a Web criada pela a macromedia (agora Adobe SYstems), que está
atualmente na versão CS3 Versões iniciais de aplicaçõa serviam como um simples editor HTML WYSIWYG ("What You See Is")
